# SATORI Forms

Core form builder plugin by SATORI for WordPress.  
Build and manage custom forms with ease.

---

## Description

SATORI Forms is a lightweight, extensible WordPress plugin designed to let you create custom forms and manage submissions.  
The plugin follows best practices with a modular, namespaced codebase.  

---

## Features

- Custom post type for forms  
- Admin UI for managing forms  
- Extensible architecture for building complex form fields (coming soon)  
- Built-in security and standards compliance  

---

## Installation

1. Upload the `satori-forms` folder to your `/wp-content/plugins/` directory  
2. Activate the plugin through the 'Plugins' menu in WordPress  
3. Start creating forms via the SATORI Forms admin menu
